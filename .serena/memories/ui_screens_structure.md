# UI 화면 구조 분석

## 전체 애플리케이션 플로우
```
홈페이지 (HomePage) 
    ↓
    ├─→ 새 프로젝트 → 작업공간 (WorkSpace)
    └─→ 기존 프로젝트 → 프로젝트 목록 (ProjectList) → 작업공간 (WorkSpace)
```

## 각 화면별 구조

### 1. 홈페이지 (HomePage)
**디자인**: 글래스모피즘 카드 2개
- **새 프로젝트** 카드: 플러스 아이콘, 시작 버튼
- **기존 프로젝트** 카드: 폴더 아이콘, 목록 보기 버튼
- **배경**: 그라데이션 배경 (blue-50 → purple-50 → pink-50)
- **로고**: codle-logo.svg 상단 중앙 배치

### 2. 프로젝트 목록 (ProjectList)
**레이아웃**: 리스트 형태의 카드 목록
- **헤더**: 홈으로 버튼 + 제목 + 설명
- **프로젝트 카드**: 이름, 생성일, 최종 수정일, 열기/삭제 버튼
- **빈 상태**: 아이콘 + 안내 문구 + 새 프로젝트 시작 버튼

### 3. 작업공간 (WorkSpace)
**3단계 플로우**:
1. **API 키 입력** (ApiKeyManager)
2. **워크플로우** (WorkflowContainer) - 5단계 진행
3. **결과 표시** (ResultDisplay)

#### 3-1. API 키 관리자 (ApiKeyManager)
- API 키 입력 폼
- 유효성 검증 버튼
- 암호화 저장 안내

#### 3-2. 워크플로우 컨테이너 (WorkflowContainer)
**디자인**: 글래스모피즘 + 그라데이션 배경
- **헤더**: 홈으로 버튼 + 제목 + 설명
- **진행 표시기**: 5개 원형 스텝 인디케이터 (연결선 포함)
- **현재 단계**: 각 단계별 컴포넌트 렌더링

##### Step 1: 기본 정보 입력 (Step1BasicInfo)
**구성 요소**:
1. **프로젝트 설정 섹션** (2열 그리드)
   - 레이아웃 모드: 고정형/스크롤형 라디오 버튼
   - 콘텐츠 모드: AI 향상/제한 모드 라디오 버튼

2. **기본 정보 입력 섹션**
   - 프로젝트 제목 (필수)
   - 대상 학습자 (필수)
   - 추가 제안사항 (선택)

3. **페이지 구성 섹션** (PageListEditor)
   - 2열 그리드 레이아웃
   - 각 페이지 카드: 번호 + 주제 + 설명
   - 페이지 추가 버튼 (점선 테두리 카드)

##### Step 2-5: 개발 진행 중
- 임시 안내 화면 (개발 진행 중 메시지)
- 이전 단계로 버튼

#### 3-3. 결과 표시 (ResultDisplay)
- 생성된 프롬프트 표시
- 복사/다운로드/편집 기능
- 리셋 버튼

## UI 디자인 시스템

### 글래스모피즘 클래스
- `.glass`: 기본 글래스 효과 (backdrop-blur-md, bg-white/10)
- `.glass-card`: 카드형 글래스 (backdrop-blur-lg, bg-white/15)
- `.glass-strong`: 강한 글래스 (backdrop-blur-xl, bg-white/20)
- `.glass-input`: 입력 필드용 (backdrop-blur-sm, bg-white/20)
- `.glass-button`: 버튼용 (bg-white/10, hover:bg-white/20)
- `.glass-button-primary`: 기본 버튼 (bg-blue-500/80)

### 색상 시스템
- **배경**: 애니메이션 그라데이션 (blue → orange → red → purple → pink)
- **텍스트**: white, white/90, white/70, white/60 (투명도 단계)
- **액센트**: blue-500/80, green-500/80, red-400 (상태별)

### 레이아웃 패턴
- **카드 기반**: 대부분의 콘텐츠가 글래스 카드 안에 배치
- **그리드 시스템**: 2열 그리드 자주 사용 (md:grid-cols-2)
- **반응형**: 모바일 우선, md 브레이크포인트 활용
- **간격**: space-y-6, gap-4, p-8 등 일관된 간격
